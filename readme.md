## ius2D-js
![](https://github.com/keicoon/require_webgl/blob/master/capture/capture.gif)

## Dependancy
```
node,
express,
browserify,
watchify,
lodash
```

## Supporting plugins
```
howler,
spine2d,
live2d
```

## How to start
* install [node.js](https://nodejs.org/en/)
* execute setup.bat
* execute run.bat
* access http://localhost:8000
